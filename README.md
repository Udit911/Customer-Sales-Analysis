# 📊 Customer Sales Analysis

🚀 A Python-based data analysis project that explores customer purchasing behavior, identifies sales trends, and generates actionable insights. The project also includes machine learning predictions, automated reporting, and an interactive dashboard.

---

## 📖 Overview
The **Customer Sales Analysis** project helps analyze transactional data to uncover patterns in customer behavior and product performance. It enables businesses to make **data-driven decisions** for marketing, inventory, and pricing strategies.

---

## ✨ Features
- 📊 Data Cleaning & Preprocessing using Pandas  
- 📈 Sales Trend Analysis & Visualization  
- 🛍️ Best-Selling Product Identification  
- 👥 Customer Spending Behavior Analysis  
- 🤖 Machine Learning for Sales Prediction  
- 📄 Automated PDF Report Generation  
- 🌐 Interactive Dashboard using Streamlit  

---

## 🛠 Tech Stack
- **Language:** Python  
- **Libraries:** Pandas, NumPy, Matplotlib, Seaborn  
- **Machine Learning:** Scikit-learn  
- **Dashboard:** Streamlit  
- **Report Generation:** FPDF  

---

## 📊 Dataset
The dataset contains:
- Customer ID  
- Purchase Date  
- Product Category  
- Quantity Purchased  
- Price per Item  
- Total Purchase Amount
